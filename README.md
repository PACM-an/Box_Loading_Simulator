## Instructions to Run the Project

1. Navigate to the project's main directory:

```
cd 3D-bin-packing
```

2. Start the 3D Bin Packing API with Python:

```
python api.py
```

3. Change to the OpenGL frontend directory:

```
cd OpenGL
```

4. Launch the PyOpenGL window:

```
python main.py
```

5. Change to the Julia backend directory:

```
cd Julia
```

6. Run the web API file in Julia:

```
julia webapi.jl
```
